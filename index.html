A Box of Thoughts

React/Redux, Express, MongoDB

mood board noun a board used by designers on which samples of various colours and textures are mounted to help in deciding which elements complement each other vision board noun a collage of pictures, text, and other items that represent and affirm one’s dreams and ambitions, created to help visualize and focus on one or more specific aspirations. From dictionary.com A Box of Thoughts is a web app that allows users to create and save digital mood or vision boards. This project includes: User Authentication Password Encryption API Calls Bootstrap MVC 2 Model CRUD Masonry CSS Once logged in or registered, users can view their created boards or make a new board. To create a new board, the user inputs a required title for the board and an optional description. Board is added to user’s collection. When “Add New Image” button is clicked, the search modal pops up. Upon submitting the search term a user has inputted, a call is made to the third-party Unsplash API. A set of images matching the search term are populated in the modal and when one is selected, it can be added to the board. User can continue searching for new images until their board is complete. As more images are added, they are sized and aligned in a grid using Masonry CSS. Bootstrap allows the app to be used on desktop or mobile while maintaining the look, feel and functionality.
